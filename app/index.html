Show Me The Numbers

Analyzing numbers one loop at a time

A javascript coding exercise using loops. Write a program that displays a set of user defined numbers. Display the even numbers in BOLD, prime numbers with a purple outline, and perfect squares with a teal outline.

Instructions

Enter a starting value and an ending value. Press the button and the values will populate in a grid. Hover over the numbers to see what their lowest divisor is or the number that is the square root.
